# PDF Reader for Mac Intel - 项目规范

## 1. 项目概述

- **项目名称**: pdf-reader-mac
- **项目类型**: 桌面应用程序 (Electron + PDF.js)
- **核心功能**: 轻量级 PDF 文件阅读器，支持翻页、缩放、目录导航
- **目标平台**: macOS Intel 处理器

## 2. 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Electron 28.x |
| PDF 渲染 | PDF.js 4.x |
| 构建工具 | electron-builder |
| 前端 | HTML5 + CSS3 + Vanilla JS |

## 3. 功能列表

### 3.1 文件操作
- [x] 通过文件对话框打开 PDF 文件
- [x] 支持拖拽打开 PDF 文件
- [x] 显示当前文件名

### 3.2 阅读导航
- [x] 上一页 / 下一页翻页
- [x] 跳转到指定页码
- [x] 显示当前页/总页数
- [x] 键盘快捷键 (← → 翻页)

### 3.3 视图控制
- [x] 放大 / 缩小
- [x] 适应页面宽度
- [x] 适应页面高度
- [x] 100% 原始大小

### 3.4 UI 组件
- [x] 顶部工具栏 (打开文件、缩放控制)
- [x] 底部状态栏 (页码信息)
- [x] 左侧页面缩略图列表 (可选)

## 4. UI/UX 设计

### 4.1 布局结构
```
┌──────────────────────────────────────┐
│  工具栏: [打开] [◀][▶] [页码] [缩放] │
├──────────────────────────────────────┤
│                                      │
│           PDF 渲染区域               │
│                                      │
│                                      │
├──────────────────────────────────────┤
│  状态栏: 第 X 页 / 共 Y 页           │
└──────────────────────────────────────┘
```

### 4.2 配色方案
- 主色: #1a1a2e (深蓝黑)
- 次要色: #16213e (深蓝)
- 强调色: #0f3460 (中蓝)
- 文字: #e8e8e8 (浅灰白)
- 边框: #333355 (蓝灰)

### 4.3 字体
- 主字体: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- 代码/数字: "SF Mono", Monaco, monospace

## 5. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| ← | 上一页 |
| → | 下一页 |
| Cmd/Ctrl + O | 打开文件 |
| Cmd/Ctrl + = | 放大 |
| Cmd/Ctrl + - | 缩小 |
| Cmd/Ctrl + 0 | 适应页面宽度 |

## 6. 构建配置

- **目标架构**: Intel x64 (darwin-x64)
- **应用名称**: PDF Reader
- **应用 ID**: com.pdfreader.mac
- **图标**: 默认 Electron 图标

## 7. 验收标准

1. 能够成功打开和显示 PDF 文件
2. 翻页功能正常，页码显示准确
3. 缩放功能正常工作
4. 键盘快捷键响应正常
5. 成功构建出 .app 文件
6. 在 macOS Intel 环境下能正常运行
