# PDF Reader for Mac Intel - 项目规范

## 1. 项目概述

- **项目名称**: pdf-reader-mac
- **项目类型**: 桌面应用程序 (Electron + PDF.js)
- **核心功能**: 轻量级 PDF 文件阅读器，支持单页翻页、书架、书签和阅读位置记忆
- **目标平台**: macOS Intel 处理器

## 2. 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Electron 28.x |
| PDF 渲染 | PDF.js 4.x |
| 构建工具 | electron-builder |
| 前端 | HTML5 + CSS3 + ES Module |
| 数据存储 | JSON 文件 (electron app.getPath) |

## 3. 功能列表

### 3.1 文件操作
- [x] 通过文件对话框打开 PDF 文件
- [x] 支持拖拽打开 PDF 文件
- [x] 显示当前文件名
- [x] 指定文件夹作为书架

### 3.2 阅读模式（单页翻页）
- [x] 上一页 / 下一页翻页
- [x] 跳转到指定页码
- [x] 显示当前页/总页数
- [x] 键盘快捷键 (← → 翻页)
- [x] 单页显示，不滚动

### 3.3 阅读位置记忆
- [x] 自动保存当前阅读页码
- [x] 重新打开同一 PDF 时自动跳转到上次位置
- [x] 防抖保存（1秒延迟）

### 3.4 书签功能
- [x] 添加书签（支持自定义名称）
- [x] 显示/隐藏书签列表
- [x] 点击书签跳转到对应页面
- [x] 删除书签
- [x] 当前页书签状态指示

### 3.5 书架功能
- [x] 选择文件夹作为书架
- [x] 自动扫描并显示文件夹中的 PDF 文件
- [x] 网格布局展示 PDF 文件
- [x] 点击书架项打开对应 PDF
- [x] 支持更换书架文件夹

### 3.6 视图控制
- [x] 放大 / 缩小
- [x] 适应页面宽度
- [x] 100% 原始大小
- [x] 窗口缩放时自动调整

### 3.7 性能优化
- [x] 加载动画提示
- [x] 按需渲染单页
- [x] PDF.js CMaps 预加载
- [x] 错误处理优化

### 3.8 UI 组件
- [x] 顶部工具栏 (打开文件、书架、缩放控制、书签)

## 5. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| ← | 上一页 |
| → | 下一页 |
| Cmd/Ctrl + O | 打开文件 |
| Cmd/Ctrl + Shift + O | 打开书架文件夹 |
| Cmd/Ctrl + B | 显示书架 |
| Cmd/Ctrl + D | 添加书签 |
| Cmd/Ctrl + = | 放大 |
| Cmd/Ctrl + - | 缩小 |
| Cmd/Ctrl + 0 | 适应页面宽度 |

## 6. 数据存储

### 6.1 存储位置
`~/Library/Application Support/pdf-reader-mac/pdf-reader-data.json`

### 6.2 数据结构
```json
{
  "readingPositions": {
    "/path/to/file.pdf": 5
  },
  "bookmarks": {
    "/path/to/file.pdf": [
      {
        "id": "1704067200000",
        "page": 5,
        "label": "第 5 页",
        "createdAt": "2024-01-01T00:00:00.000Z"
      }
    ]
  }
}
```

4. 缩放功能正常工作
5. 书架能正确扫描并显示文件夹中的 PDF
6. 点击书架项能打开对应 PDF
7. 阅读位置记忆功能正常
8. 书签添加、删除、跳转功能正常
9. 加载动画正常显示
10. 成功构建出 .app 文件
11. 在 macOS Intel 环境下能正常运行
