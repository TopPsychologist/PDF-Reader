# PDF Reader for Mac Intel - 项目规范

## 1. 项目概述

- **项目名称**: pdf-reader-mac
- **项目类型**: 桌面应用程序 (Electron + PDF.js)
- **核心功能**: 轻量级 PDF 文件阅读器，支持单页翻页、书架、书签、目录和阅读位置记忆
- **目标平台**: macOS Intel 处理器

## 2. 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Electron 28.x |
| PDF 渲染 | PDF.js 4.x |
| 构建工具 | electron-builder |
| 前端 | HTML5 + CSS3 + ES Module |
| 数据存储 | JSON 文件 (electron app.getPath) |

## 3. 功能列表

### 3.1 文件操作
- [x] 通过文件对话框打开 PDF 文件
- [x] 支持拖拽打开 PDF 文件
- [x] 显示当前文件名
- [x] 指定文件夹作为书架
- [x] 书架路径持久化保存

### 3.2 阅读模式（单页翻页）
- [x] 上一页 / 下一页翻页
- [x] 跳转到指定页码
- [x] 显示当前页/总页数
- [x] 键盘快捷键 (← → 翻页)
- [x] 单页显示，不滚动
- [x] 适应窗口高度显示

### 3.3 阅读位置记忆
- [x] 自动保存当前阅读页码
- [x] 重新打开同一 PDF 时自动跳转到上次位置
- [x] 防抖保存（1秒延迟）

### 3.4 书签功能
- [x] 添加书签（支持自定义名称）
- [x] 显示/隐藏书签列表
- [x] 点击书签跳转到对应页面
- [x] 删除书签
- [x] 当前页书签状态指示

### 3.5 目录功能
- [x] 解析 PDF 大纲/书签作为目录
- [x] 显示/隐藏目录侧边栏
- [x] 点击目录项跳转到对应页面
- [x] 目录加载状态提示

### 3.6 书架功能
- [x] 选择文件夹作为书架
- [x] 自动扫描并显示文件夹中的 PDF 文件
- [x] 网格布局展示 PDF 文件
- [x] 书架封面预览（第一页缩略图）
- [x] 显示阅读进度百分比
- [x] 点击书架项打开对应 PDF
- [x] 支持更换书架文件夹

### 3.7 视图控制
- [x] 放大 / 缩小
- [x] 适应页面宽度
- [x] 适应窗口高度
- [x] 窗口缩放时自动调整

### 3.8 UI 组件
- [x] 顶部工具栏 (打开文件、书架、返回书架、缩放控制、目录、书签)
- [x] 底部状态栏 (页码信息、书签状态)
- [x] 书架视图 (网格布局、封面预览、阅读进度)
- [x] 书签侧边栏
- [x] 目录侧边栏

## 5. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| ← | 上一页 |
| → | 下一页 |
| Cmd/Ctrl + O | 打开文件 |
| Cmd/Ctrl + Shift + O | 打开书架文件夹 |
| Cmd/Ctrl + B | 返回书架 |
| Cmd/Ctrl + D | 添加书签 |
| Cmd/Ctrl + = | 放大 |
| Cmd/Ctrl + - | 缩小 |
| Cmd/Ctrl + 0 | 适应页面宽度 |
| Cmd/Ctrl + 1 | 适应窗口高度 |

## 6. 数据存储

### 6.1 存储位置
`~/Library/Application Support/pdf-reader-mac/pdf-reader-data.json`

### 6.2 数据结构
```json
{
  "readingPositions": {
    "/path/to/file.pdf": 5
  },
  "bookmarks": {
    "/path/to/file.pdf": [
      {
        "id": "1704067200000",
        "page": 5,
        "label": "第 5 页",
        "createdAt": "2024-01-01T00:00:00.000Z"
      }
    ]
  },
  "shelfFolder": "/path/to/shelf/folder"
}
```

## 8. 验收标准

1. 能够成功打开和显示 PDF 文件
2. 单页显示，适应窗口高度
3. 翻页功能正常，页码显示准确
4. 缩放功能正常工作（适应宽度/高度）
5. 书架能正确扫描并显示文件夹中的 PDF
6. 书架封面预览正常显示
7. 书架阅读进度正常显示
8. 书架路径持久化保存
9. 书架打开后保留，无需重复选择
10. 点击书架项能打开对应 PDF
11. 目录功能正常（解析并跳转）
12. 阅读位置记忆功能正常
13. 书签添加、删除、跳转功能正常
14. 返回书架按钮正常工作
15. 成功构建出 .app 文件
16. 在 macOS Intel 环境下能正常运行
